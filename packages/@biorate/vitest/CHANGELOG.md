# Changelog

## 1.0.0

### Features

- Initial release
- OOP test decorators for Vitest
- Full Allure integration
- Support for test lifecycle hooks (beforeAll, afterAll, before, after)
- Test modifiers (skip, only, todo, timeout, repeats)
- Allure annotations (epic, feature, story, severity, owner, tags, labels, links, issues)
- Scenario pattern for reusable test steps
- Suite options (mode, timeout, retries)
- TypeScript support with full type safety
- Clean modular architecture
